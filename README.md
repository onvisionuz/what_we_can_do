# What We Can Do

A showcase page presenting our work, cases, and capabilities.

## Tech

Plain HTML, CSS, and JavaScript — no build tools required.

## Project Structure

```
├── index.html       # Main entry point
├── styles.css       # Global styles
├── app.js           # Main application logic
├── case-data.js     # Case content data
├── case-media.js    # Case media references
└── assets/          # Images and static assets
```

## Deployment

Hosted on Vercel. Every push to `main` triggers an automatic redeploy.
